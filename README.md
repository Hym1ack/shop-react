# Интернет-магазин на React.

deploy : https://shop-react-five.vercel.app/

Стэк :

* React 18.2.0
* React router dom 6.4.0
* Redux 8.0.2
* Redux Toolkit 1.8.5
* Swiper 8.4.2
* Firebase 9.10.0
* Vercel

### `npm i`

Установка зависимостей

### `npm run start`

Запуск в dev режиме
